# Wallet Service

Приложение на Spring Boot для управления кошельками с поддержкой конкурентных операций.

---

## Описание

Проект предоставляет REST API для работы с кошельками:

- **POST /api/v1/wallet** – выполнить операцию с кошельком (DEPOSIT или WITHDRAW)
- **GET /api/v1/wallets/{walletId}** – получить текущий баланс кошелька

Особенности:

- Потокобезопасная обработка операций с использованием транзакций и блокировки записи (`PESSIMISTIC_WRITE`)
- Обработка ошибок: несуществующий кошелёк, недостаточно средств, некорректные данные
- Конфигурация через переменные окружения
- База данных PostgreSQL с миграциями Liquibase
- Полностью покрыто тестами, включая стресс-тест для конкурентных операций

---

## Технологии

- Java 17
- Spring Boot 3
- Spring Data JPA
- PostgreSQL
- Liquibase
- Docker / Docker Compose
- JUnit 5
